##  Project Overview

CoreTech Innovations is a professional technology company website built with modern web technologies. The website features a clean, minimal design with smooth animations and full responsiveness across all devices. The project demonstrates front-end development skills including responsive design, interactive elements, and modern UI/UX principles.

**Live Demo**: (https://rajamuhammadarbab.github.io/CoreTech_WebFinal_Raja_Muhammad_Arbab/)  
**GitHub Repository**: (https://github.com/RajaMuhammadArbab/CoreTech_WebFinal_Raja_Muhammad_Arbab.git)

### Key Highlights
-  Dark/Light mode with system preference detection
-  Fully responsive design (mobile, tablet, desktop)
-  Fast loading with optimized performance
-  Modern, professional design with smooth animations
-  Accessibility-focused development

##  Technologies Used

### Frontend Stack
- **HTML5** - Semantic markup and structure
- **Tailwind CSS** - Utility-first CSS framework
- **JavaScript (ES6+)** - Interactive functionality
- **Font Awesome** - Icons and UI elements

### Development Tools
- **Git** - Version control
- **VS Code** - Code editor
- **Chrome DevTools** - Debugging and testing
- **Live Server** - Local development

### External Services
- **Tailwind CSS CDN** - Styling framework
- **Font Awesome CDN** - Icons library
- **Unsplash** - Stock images (for demo)

## Features Implemented

### Core Features
- ✅ **5 Complete Pages**: Home, About, Services, Portfolio, Contact
- ✅ **Responsive Design**: Mobile-first approach for all screen sizes
- ✅ **Dark/Light Mode**: Toggle with system preference detection
- ✅ **Professional Navigation**: Mobile hamburger menu with smooth transitions
- ✅ **Hero Section**: Eye-catching banner with call-to-action buttons
- ✅ **Services Showcase**: Animated service cards with hover effects
- ✅ **Team Profiles**: Team member sections with social links
- ✅ **Portfolio Filtering**: JavaScript-powered project categorization
- ✅ **Contact Form**: Front-end validation with error handling
- ✅ **SEO Optimization**: Semantic HTML and proper meta tags

### Interactive Features
- ✅ **Smooth Scrolling**: Enhanced navigation experience
- ✅ **Hover Animations**: CSS transitions and transforms
- ✅ **Form Validation**: Real-time feedback and error messages
- ✅ **Portfolio Filtering**: Category-based project filtering
- ✅ **Mobile Menu**: Touch-friendly navigation
- ✅ **Theme Persistence**: localStorage for user preferences
- ✅ **Visitor Counter**: Simple analytics feature

### Technical Features
- ✅ **Clean Code Structure**: Modular and maintainable
- ✅ **Separate Files**: Organized CSS and JavaScript
- ✅ **Semantic HTML**: Accessibility and SEO friendly
- ✅ **Performance Optimized**: Minimal external dependencies
- ✅ **Cross-Browser Compatible**: Works on all modern browsers
- ✅ **Error Handling**: Graceful fallbacks and validation
